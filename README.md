# Spring Boot + ScalikeJDBC4J + Lombok sample project

## Prerequisites

- Java SE 8
- Scala 2.11
- Maven 3

## How to build

    $ mvn compile

## How to run

    $ mvn exec:java

Open a browser on your machine and open [http://localhost:8080/companies/](http://localhost:8080/companies/)

## ScalikeJDBC4J

https://github.com/scalikejdbc/scalikejdbc4j
